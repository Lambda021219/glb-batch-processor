# GLB 自动化拓扑与烘焙工具 (Blender 插件)

将高模 GLB 文件批量处理为低模 + 烘焙贴图的 Blender 插件。一键完成 Quad Remesher 重拓扑、智能 UV 展开、Cycles GPU 烘焙漫射/法线贴图，按模型分文件夹导出。

## 功能特性

- 🚀 批量导入高模 .glb 文件并自动合并网格
- 🔧 Quad Remesher 自动重拓扑（可调目标面数）
- 📐 智能 UV 投射
- 🎨 Cycles GPU 烘焙（漫射 + 可选法线贴图）
- 📁 按模型名自动创建子文件夹输出
- 📦 多格式导出：GLB / FBX / OBJ / USD / STL
- 📊 实时进度条 + 详细日志
- 🌐 中英双语界面
- ⏹ 随时停止处理
- 📂 单文件处理模式（用于测试参数）

## 系统要求

| 项目 | 最低版本 |
|------|----------|
| Blender | 4.0+ |
| Quad Remesher | 1.23+ |
| GPU | 推荐 NVIDIA RTX (OptiX) / AMD (HIP) / Apple (Metal) |

> 💡 没有 GPU 也可用 CPU 烘焙，但速度较慢。

## 安装方法

1. 从 [Releases](https://github.com/Lambda021219/glb-batch-processor/releases) 下载 `glb-batch-processor.zip`
2. 打开 Blender → **Edit** → **Preferences** → **Add-ons** → 点击 **Install**
3. 选择下载的 `.zip` 文件 → 点击 **Install Add-on**
4. 在插件列表中搜索 "GLB" → 勾选启用 **GLB 自动化拓扑与烘焙工具**
5. 在 3D Viewport 中按 **N 键** → 侧边栏出现 **"GLB处理"** 标签页

## 使用方法

### 批量处理

1. 在 **📁 输入输出** 区域设置输入文件夹（存放高模 .glb 文件）和输出文件夹
2. 根据需要调整 **🔧 重拓扑设置**（目标面数、超时等）
3. 调整 **🎨 烘焙设置**（贴图分辨率、是否烘焙法线、贴图格式）
4. 调整 **📦 导出设置**（导出格式、是否平铺输出）
5. 点击 **▶ 开始批量处理**
6. **处理期间请勿操作 Blender**，等待弹窗提示完成

### 单文件处理（测试用）

点击 **处理单个文件...**，选择 GLB 文件，即可用当前设置处理一个模型。

### 输出结构

默认每个模型创建独立子文件夹：

```
输出文件夹/
├── 模型A/
│   ├── 模型A_low.glb          # 低模 GLB 文件
│   ├── 模型A_diffuse.png      # 漫射贴图
│   └── 模型A_normal.png       # 法线贴图 (如开启)
├── 模型B/
│   ├── 模型B_low.glb
│   ├── 模型B_diffuse.png
│   └── 模型B_normal.png
└── batch_process_log.txt      # 处理日志
```

> 💡 在导出设置中勾选 **平铺输出** 可恢复旧版行为（所有文件输出到根目录）。

## 面板参数说明

### 输入输出
| 参数 | 说明 |
|------|------|
| 输入文件夹 | 存放高模 .glb 文件的目录 |
| 输出文件夹 | 处理结果输出目录（默认为输入目录下的"输出"文件夹）|

### 重拓扑设置
| 参数 | 默认值 | 说明 |
|------|--------|------|
| 目标面数 | 5000 | Quad Remesher 重拓扑目标四边形数量 |
| 自适应面数 | 关闭 | 让 Quad Remesher 自动判断合适的面数 |
| 超时时间 | 300秒 | 等待 Quad Remesher 外部引擎完成的最长时间 |

### 烘焙设置
| 参数 | 默认值 | 说明 |
|------|--------|------|
| 贴图分辨率 | 1024 | 烘焙贴图的像素尺寸 |
| 包裹笼挤出 | 0.10m | 烘焙 Cage 向外挤出的距离 |
| 烘焙法线贴图 | 关闭 | 是否额外烘焙一张法线贴图 |
| 单独保存贴图 | 开启 | 将贴图保存为独立文件 |
| 贴图格式 | PNG | 贴图文件格式（PNG 无损 / JPEG 有损）|

### 导出设置
| 参数 | 默认值 | 说明 |
|------|--------|------|
| 导出格式 | GLB | GLB / glTF Separate / FBX / OBJ / USD / STL |
| 平铺输出 | 关闭 | 不创建子文件夹，所有文件直接输出到根目录 |
| 完成后自动关闭 | 关闭 | 处理完毕后自动退出 Blender |

> ⚠️ **FBX/OBJ/STL 格式不支持内嵌贴图**，使用这些格式时请务必在烘焙设置中开启「单独保存贴图」。

## 常见问题

### Q: 点击开始后提示 "未检测到 Quad Remesher 插件"

**A:** 请先安装 Quad Remesher。购买地址: [exoside.com](https://exoside.com/quadremesher/)。安装后需在 Blender 偏好设置中启用。

### Q: 重拓扑一直显示"等待中"，最后超时

**A:** 
1. 检查 Quad Remesher 是否正常激活（尝试在 Blender 中手动使用一次）
2. 增大超时时间（复杂模型可能需要更长时间）
3. 在系统托盘中检查 Quad Remesher 外部引擎是否运行

### Q: 烘焙报错或贴图全黑

**A:**
1. 确保高模和低模在场景中可见（未隐藏）
2. 检查 GPU 驱动是否为最新
3. 尝试在偏好设置中切换 Cycles 渲染设备（OptiX → CUDA）
4. 增大包裹笼挤出值（默认 0.1m 可能不够）

### Q: 导出的 GLB 没有贴图

**A:** 检查烘焙设置中 **"单独保存贴图"** 是否开启。贴图会保存在与 GLB 相同的子文件夹中。

### Q: 处理多个文件时内存不足

**A:** 每个文件处理完后会自动清理场景。如果仍内存不足，建议减少贴图分辨率（如 512），或关闭法线烘焙。
